## Strings
1. Single Quotes
```JavaScript
const design = 'Heroku Design Team';
```

Adding an additional `'` by escaping using `/`
```JavaScript
const sentence = 'She/'s our manager.'
```

2. Double Qoutes
```JavaScript
const product = "Heroku Product Design";
```
3. Backticks
```JavaScript
const development = `Heroku Frontend Design`;
```
